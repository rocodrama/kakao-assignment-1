# kakao-assignment-1

## My Todo List

> Vanilla JS로 만든 날짜별 할 일 관리 앱

---

### 주요 기능

- **Todo CRUD** — 할 일 추가 / 수정 / 삭제 / 완료 처리
- **상태별 필터링** — 전체 / 진행 중 / 완료 탭으로 분류
- **일간 뷰** — 선택한 날짜의 Todo만 표시
- **로컬스토리지** — 새로고침해도 데이터 유지
- **주간 뷰** — 이번 주 날짜를 가로로 표시, 이전·다음 주 이동
- **달력 팝업** — 달력 아이콘 클릭으로 원하는 날짜로 이동
- **드래그앤드롭** — 마우스로 항목을 잡아 순서 변경

---

### 기술 스택

| 분류 | 사용 기술 |
|------|-----------|
| 마크업 | HTML5 |
| 스타일 | CSS3 |
| 로직 | Vanilla JavaScript (ES6+) |
| 저장 | LocalStorage |

라이브러리·프레임워크 없이 순수 Vanilla JS로 구현했습니다.

---

### 파일 구조

```
main/
├── index.html   # 앱 레이아웃
├── style.css    # 스타일
└── app.js       # 전체 로직
```

별도의 설치나 빌드 없이 `index.html`을 브라우저에서 열면 바로 실행됩니다.
